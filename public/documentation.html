Welcome to my Movie Api Documentation Page

This Api is being designed to return a list of movies with individial data sets about each movie.
It's also being designed to return data about different genres and directors.
It should allow users to register with a username, password, email and birthday.
Allow each individual user to make lists of their favorite movies or watch later list and be able to remove movies from their list.
It should also allow users to delete their account or de-register, deleting their data from the database.
