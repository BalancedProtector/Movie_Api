Welcome to my Movie Api Documentation Page

List of Get request URL's

  • Home Page

    Method: .get

    URL: http://localhost:8080/

    URL: http://localhost:8080/index.html

    The homepage of the app that everyone is guaranteed to see

  • Get all movies

    Method: .get

    URL: http://localhost:8080/public/movies

    Returns the list of all the movies in the API

  • Get one movie

    Method: .get

    URL: http://localhost:8080/public/movies/:title

    Returns a single movie, searched by the user, if it's in the API

  • Get the movies in one Genre

    Method: .get

    URL: http://localhost:8080/movies/:genre

    Returns a list of all movies within the selected genre

  • Get the movies from one Director

    Method .get

    URL: http://localhost:8080/movies/:director

    Returns a list of movies directed by a single director

  • Get all the movies from one Year

    Method: .get

    URL: http://localhost:8080/movies/:year

    Returns a list of movies all released in the same year

  • Get the movies in a given genre from one year

    Method: .get

    URL: http://localhost:8080/movies/:genre/:year

    Returns a list of movies that contain the same genre and released in a single year, both specified by the user

  • Get this very documentation list

    Method: .get

    URL: http://localhost:8080/documentation

    URL: http://localhost:8080/public/documentation

    Returns this documentation page

  • Add a new movie to the API

    Method: .post

    URL: http://localhost:8080/movies

    Adds a new User Specified Movie using the following format:

    {"title": " ", "year": " ", "genre": " ", "director": " ", "thumbnail": " 'url here' "}
