Welcome to my Movie Api Documentation Page

List of Get request URL's

  • Home Page

    Method: .get

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/

    URL: /index.html

    The homepage of the app that everyone is guaranteed to see

  • Get all movies

    Method: .get

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/movies

    Returns the list of all the movies in the API

  • Get one movie

    Method: .get

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/movies/:Title

    Returns a single movie, searched by the user, if it's in the API

  • Get the Genre by name

    Method: .get

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/genres/:GenreName

    Returns a list of all movies within the selected genre

  • Get the a director by name

    Method .get

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/directors/:DirectorName

    Returns a list of movies directed by a single director

  • Users by Name

    Method: .get

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/users/:Name

    Retrieves all information on one user

    Specified by Name or their Username

  • Get this very documentation list

    Method: .get

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/documentation

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/public/documentation

    Returns this documentation page

  • Add a new movie to the API

    Method: .post

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/movies

    Adds a new Movie using the following JSON format:

    {"Title": " ", "Year": Date , "Genre": " ", "Director": " ", "Thumbnail": " 'url here' "}

  • Register a new User

    Method: .post

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/users

    Adds a New User using the following JSON format:

    {"Name": "" , "Password": "" , "Email": "", "Birthday": "Date" }

  • Login an Existing User

    Method: .post

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/login?Name=(name)&Password=(password)

    This logs in an existing user using the base URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com and adding Name and Password params.

    Name is short for Username in this app, not standard understandably but I had to grab your attention somehow, with Password being standard for Password, this app uses hashed passwords for added security.

  • Add a movie to a users favorite movies array

    Method: .post

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/users/:Name/movies/:MovieID

    Adds a movie to a users favorite movies array

  • Updates a Users Data

    Method: .put

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/users/:Name

    Updates a users information in the following JSON format:

    {Name : { type : String, required : true}}, {Password:{ type : String, required : true}}, {Email : { type : String, required : true}}, {Birthday: Date}, {favoriteMovies:[{ type : mongoose.Schema.Types.ObjectId, ref : 'Movie'}]}

  • Deletes a movie from a users favorite movies array

    Method: .delete

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/users/:Name/movies/:MovieID

    Deletes a movie from a users favorite movies array

  • Delete a User

    Method: .delete

    URL: https://mind-theatre-api-dc69e2dcb161.herokuapp.com/users/:Name

    Deletes a user from the API
